# SMUtime

  SMUtime은 상명대학교 학생들을 위한 편의기능을 제공하는 커뮤니티이다.
  
## SMUtime의 주요 기능

    - 회원가입/로그인
    - 게시판
    - 시간표
    - 채팅
    - 셔틀버스시간표
    - 버스시간표
    - 택시 모집

## SMUtime의 차이점
SMUtime은 다른 어플들과는 다르게 상명대학교 학생들만을 위한 어플이다. 따라서 상명대학교 학생들을 위한 차별화된 편의 기능을 제공하고 있다. 우선 회원가입을 진행할때 상명대학교 이메일을 제외한 다른 이메일은 사용할 수 없도록 구현하였다.
