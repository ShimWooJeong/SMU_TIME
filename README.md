# SMUtime

  SMUtime은 상명대학교 학생들을 위한 편의기능을 제공하는 커뮤니티이다.
  
## SMUtime의 주요 기능

    - 회원가입/로그인
    - 게시판
    - 시간표
    - 채팅
    - 셔틀버스시간표
    - 버스시간표
    - 택시 모집
